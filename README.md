## Açıklama

Astrology API projesi [Nest](https://github.com/nestjs/nest) framework'ünü ve typescript kullanarak yapılmıştır

## Yüklemek İçin

```bash
$ yarn install
```

## Çalıştırmak İçin

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
